# Sigorta CRM - UI & Theming Rules

## Design System
- **Primary Color**: Emerald (Tailwind: `emerald-600`, `emerald-700`).
- **Secondary**: Teal/Gray for backgrounds.
- **Font**: Inter (Google Fonts).

## Component Guidelines
- Use `shadcn/ui` based components in `@/components/ui`.
- Avoid hardcoded hex values; use Tailwind utility classes.
- Ensure forms have validation feedback (React Hook Form).
